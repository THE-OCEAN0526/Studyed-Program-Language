# 函式(function)
- 數學的函式(function)

  〇 一元多項式函式 

    ■ f(x) = x + 3
    
    ■ f(4) = 4 + 3 = 7

〇 可想成是另一段寫好的程式碼

- 函式定義

  一個函式可接受零到多個變數, 但只能有一個回傳值

  函式內不能再定義其他函式

![image](https://user-images.githubusercontent.com/122972916/217112377-8c9cfac2-a578-4606-830e-51082b67c652.png)

```
/*回傳值的資料型態 函式名稱(參數的資料型態 參數名稱, ...){

  /*程式片段*/
  
  return 回傳值;
}*/

// 函式呼叫
// 函式名稱(引數值, ...)
-------------------------------------------------------
int f(int x){
  return x + 3;
}

int main() {
  printf("%d\n", f(4));
  return 0;
}
```
